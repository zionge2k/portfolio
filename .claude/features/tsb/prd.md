# PRD: TSB (Today Stock Briefing) 아카이브

## 배경

`~/projects/stock`의 자동 브리핑 파이프라인(launchd 08:00/18:00)이 매 거래일
자체 CSS를 가진 독립 HTML 카드를 생성한다. 이 산출물을 포트폴리오에서
`/tsb` 메뉴로 열람할 수 있어야 한다.

## 요구사항

- `/tsb`: 발행된 브리핑 날짜 아카이브 목록 (최신순)
- `/tsb/{date}`: 해당 날짜 브리핑 원본 열람
- 발행 주체는 외부 파이프라인 — `content/stock-briefing/{date}.html`을
  commit·push하면 Vercel 재배포로 반영된다 (포트폴리오 쪽 수동 작업 없음)
- 브리핑 카드의 자체 디자인을 훼손하지 않는다

## 비요구사항

- 브리핑 내용 검색·태그 분류 (블로그와 다른 성격의 데이터)
- 포트폴리오 안에서의 브리핑 생성·수정
